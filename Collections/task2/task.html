In this task you will create a Counter object that keeps track of the count of objects in an array of objects.

Counter

  • Has a private variable counts of type HashMap with a generic key and an Integer value.
  • The constructor of Counter should take in an array of generic objects. Then, it should loop through the objects and:
    • Check if the object exists in the HashMap
    • If it exists, increase its value by one.
    • If it doesn't exist, then insert the key with a value of 1
  • mostFrequent should return the key that corresponds to the element with the highest count (or value). That is, the most frequent element.
  • getCount takes in a key and returns its value. It is what is called a wrapper for a method that already exists in a HashMap. By providing our own method we restrict access to the HashMap by other developers.

Task

You need to figure out how to create a Counter for the words of the text variable so that everything compiles.
